# Plan 01-02: Frontend UI — Threshold Configuration in SensorCard

## Goal

Thêm `min_value`/`max_value` vào TypeScript types. Tạo `ThresholdEditModal` component. Thêm nút chỉnh ngưỡng vào `SensorCard` (chỉ hiện với SENSOR devices). Sau khi lưu, cập nhật UI để hiển thị dải ngưỡng trên progress bar.

## Context

- `DeviceResponse` trong `types/index.ts` chưa có `min_value`/`max_value` — cần thêm (optional)
- `DeviceWithModel` extends `DeviceResponse` — sẽ tự kế thừa các trường mới
- `SensorCard.tsx` đã có pattern `RenameDeviceModal` với nút edit hover — dùng cùng pattern cho threshold
- API call dùng `api` từ `@/lib/axios` — pattern: `api.patch('/devices/{id}/threshold', body)`
- Progress bar cuối SensorCard hiện dùng `width: ${Math.min(Math.max(value, 0), 100)}%` (hardcode 0-100) — cần dùng min/max thực tế
- UI dùng Tailwind CSS và Material Symbols Outlined icons

<action>
Tạo file mới `frontend/src/components/devices/ThresholdEditModal.tsx`:

```tsx
import { useState, useEffect } from 'react'
import toast from 'react-hot-toast'
import api, { getApiErrorMessage } from '@/lib/axios'

interface ThresholdEditModalProps {
  isOpen: boolean
  onClose: () => void
  deviceId: string
  deviceName: string
  currentMin: number | null
  currentMax: number | null
  unit: string
  onSuccess: () => void
}

export default function ThresholdEditModal({
  isOpen, onClose, deviceId, deviceName, currentMin, currentMax, unit, onSuccess
}: ThresholdEditModalProps) {
  const [minValue, setMinValue] = useState<string>(currentMin?.toString() ?? '')
  const [maxValue, setMaxValue] = useState<string>(currentMax?.toString() ?? '')
  const [isSaving, setIsSaving] = useState(false)

  useEffect(() => {
    if (isOpen) {
      setMinValue(currentMin?.toString() ?? '')
      setMaxValue(currentMax?.toString() ?? '')
    }
  }, [isOpen, currentMin, currentMax])

  if (!isOpen) return null

  const handleSave = async () => {
    const min = minValue === '' ? null : parseFloat(minValue)
    const max = maxValue === '' ? null : parseFloat(maxValue)

    if (min !== null && max !== null && min >= max) {
      toast.error('Min value must be less than max value')
      return
    }

    setIsSaving(true)
    try {
      await api.patch(`/devices/${deviceId}/threshold`, {
        min_value: min,
        max_value: max,
      })
      toast.success('Threshold updated successfully')
      onSuccess()
      onClose()
    } catch (error) {
      toast.error(getApiErrorMessage(error))
    } finally {
      setIsSaving(false)
    }
  }

  return (
    <div className="fixed inset-0 bg-black/40 backdrop-blur-sm flex items-center justify-center z-50 p-4">
      <div className="bg-surface rounded-3xl shadow-2xl w-full max-w-sm p-6 animate-fade-in">
        <h2 className="font-headline text-lg font-bold text-on-surface mb-1">
          Configure Threshold
        </h2>
        <p className="text-sm text-on-surface-variant mb-6">
          Set valid value range for <span className="font-semibold">{deviceName}</span>
        </p>

        <div className="space-y-4">
          <div>
            <label className="label-text mb-1.5 block">
              Min Value {unit && <span className="text-on-surface-variant">({unit})</span>}
            </label>
            <input
              type="number"
              value={minValue}
              onChange={(e) => setMinValue(e.target.value)}
              placeholder="No minimum"
              className="input w-full"
              id="threshold-min-input"
            />
          </div>

          <div>
            <label className="label-text mb-1.5 block">
              Max Value {unit && <span className="text-on-surface-variant">({unit})</span>}
            </label>
            <input
              type="number"
              value={maxValue}
              onChange={(e) => setMaxValue(e.target.value)}
              placeholder="No maximum"
              className="input w-full"
              id="threshold-max-input"
            />
          </div>
        </div>

        <div className="flex gap-3 mt-6">
          <button
            onClick={onClose}
            className="btn-secondary flex-1"
            disabled={isSaving}
          >
            Cancel
          </button>
          <button
            onClick={handleSave}
            className="btn-primary flex-1"
            disabled={isSaving}
            id="threshold-save-button"
          >
            {isSaving ? 'Saving...' : 'Save'}
          </button>
        </div>
      </div>
    </div>
  )
}
```
</action>

{value !== undefined && (
  <div className="mt-4 w-full bg-surface-container-low h-1.5 rounded-full overflow-hidden">
    <div
